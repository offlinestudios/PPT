# photos/gbp/

This folder holds images intended for upload to the **Google Business Profile** for Passport Photo Toronto.

Each image placed here is served as a direct, publicly accessible file URL:

```
https://www.passportphototoronto.com/photos/gbp/<filename>.jpg
```

## File requirements

| Requirement | Detail |
|---|---|
| Format | JPG or PNG |
| Minimum size | 250×250 px (aim for 1200 px wide or larger) |
| File size | Keep reasonable — avoid uncompressed originals |
| Filename | Lowercase, hyphenated, no spaces |

## Good filename example

```
passport-photo-toronto-2026-04-15-01.jpg
```

## Notes

- This folder is **not linked from any page** on the website.
- It is intentionally excluded from search engine indexing via `robots.txt`.
- Drop images here, commit, and push — they will be live and directly fetchable by Google.
